# Exportações do Brasil para os Estados Unidos (2000–2024)

Este repositório apresenta uma análise dos produtos mais exportados pelo Brasil para os Estados Unidos ao longo das últimas duas décadas. A base utilizada é dos dados oficiais do Comex Stat, permitindo reproduzir padrões reais da pauta exportadora brasileira.

## Conteúdo do repositório
- `notebooks/analise.ipynb`: notebook com análise exploratória, gráficos e interpretação econômica.

## Objetivo
Demonstrar, de maneira clara e acessível, como evoluíram os principais grupos de produtos exportados pelo Brasil para os EUA, destacando tendências estruturais e mudanças na composição da pauta ao longo do tempo.

## Principais conclusões
Ao longo do período analisado:
- produtos industriais (máquinas, veículos e equipamentos elétricos) ganharam importância;  
- óleos combustíveis se mantiveram entre os itens líderes, especialmente após 2010;  
- os EUA permanecem como um parceiro chave para bens de maior intensidade tecnológica.

Este projeto é ideal para entrevistas e demonstrações de habilidades em análise de dados aplicada à economia internacional.
